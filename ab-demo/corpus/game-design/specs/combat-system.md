## Turn order

Each combat round proceeds in this **fixed order**:

1. **Initiative roll** — highest agility acts first among living combatants
2. **Player party** — left-to-right slot order
3. **Enemies** — ascending threat tier, then random within tier

A combatant with 0 HP is skipped for the rest of the encounter.

## Stamina

Actions consume stamina; see [[stamina]] for regeneration rules outside combat.
