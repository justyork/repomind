## Vision

v2+v3 made RepoMind a credible **in-repo knowledge workspace** (links, formats, images, domains).
v4 must **prove the thesis** (agents query better than raw markdown) and **ship distribution**
before building team write workflows.

### 12-month ideal

`npx repo-mind init` → dogfood on real project → A/B shows fewer tokens + fewer hallucinations →
npm adoption → agent proposes drafts via MCP → human merges via PR.
